## Features
- Search and play music while app is in background
- User Authentication
- Browse other user's profiles and playlists
- Upload new music

## Technologies
- Jetpack Compose
- Kotlin Coroutines & Flows
- Dagger-Hilt
- Retrofit
- JUnit
